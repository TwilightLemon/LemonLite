# Privacy Policy

**Effective Date:** 2025-12-01

This Privacy Policy describes how **Lemon Lite** ("the Application", "we", "our", or "us") handles user data. By using the Application, you agree to the terms described in this Privacy Policy.



## 1. Information We Collect

### 1.1 System Media Transport Controls (SMTC)

The Application reads **SMTC (System Media Transport Controls)** data from the operating system in order to:

* Detect the currently playing media session
* Sync and display corresponding lyrics
* Access only the **supported media session metadata** allowed by the user

The Application **does not** play any media itself.
The media content originates from applications the user has chosen to allow via SMTC.

### 1.2 Local Configuration Files

The Application may read/write **local configuration files** (e.g., app settings). No personal data is stored.

### 1.4 No Personal Information Collected

The Application does **not** collect, store, or transmit any personally identifiable information (PII).
User login or account creation is not required.

How Information Is Used

No information is shared with third parties.
